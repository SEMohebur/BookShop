# 📚 Online Book Shop

A modern **MERN Stack Online Book Shop** built with **React, Node.js, Express.js, MongoDB, Firebase Authentication, Tailwind CSS, and DaisyUI**.

The application provides a smooth and responsive experience for browsing books, managing shopping carts, secure authentication, and role-based book management. It also demonstrates real-world CRUD operations, Firebase Authentication, Docker containerization, and GitHub Actions CI integration.

# 🐳 Docker Support

This project has been fully containerized using **Docker** and **Docker Compose**.

### Features

* Dockerized React Application
* Docker Compose Configuration
* Consistent Development Environment
* Easy Project Setup on Any Machine

### Run with Docker

```bash
docker compose up --build
```

### Stop Docker Containers

```bash
docker compose down
```

# ⚙️ Continuous Integration (CI)

This project uses **GitHub Actions** for Continuous Integration.

### CI Pipeline

Automatically runs on every push to the **main** branch.

Pipeline Steps

* Checkout Repository
* Setup Node.js
* Install Dependencies
* Build Application
* Verify Successful Build

This helps ensure that every commit builds successfully before deployment.

# 📂 Installation

## Clone Repository

```bash
git clone https://github.com/SEMohebur/BookShop.git
```

---

## Go to Project

```bash
cd BookShop
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Build Project

```bash
npm run build
```
